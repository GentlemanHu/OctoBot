# Partager et importer vos profils OctoBot

## Comment partager un profil OctoBot

Pour partager un profil OctoBot, ouvrez la configuration de votre profil, cliquez sur "Edit profiles", accédez à la vue d'ensemble de votre profil et cliquez sur "Share".
 

![share octobot profile](/images/guides/configuration/share-octobot-profile.png)

Votre profil sera téléchargé sous forme de dossier compressé (zip), prêt à être importé par un autre OctoBot.

## Comment importer un profil OctoBot

Pour importer un profil OctoBot, ouvrez la configuration de votre profil, cliquez sur "Edit profiles", accédez à la vue d'ensemble d'un profil et cliquez sur "Import". OctoBot vous invitera alors à sélectionner le profil à importer et l'importera.

![import octobot profile](/images/guides/configuration/import-octobot-profile.png)

Votre profil importé sera ensuite disponible à côté de vos autres profils, prêt à être utilisé et modifié.
