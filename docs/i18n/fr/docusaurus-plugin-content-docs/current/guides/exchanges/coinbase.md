Selectionner l'échange `coinbase` pour trader sur Coinbase avec OctoBot.

## Permissions requises pour un compte Coinbase
Pour trader sur Coinbase avec OctoBot, les permissions suivantes sont nécessaires sur votre API Key.
- `View`
- `Trade`
