# Trader sur Bybit

## Supporté par OctoBot

:::info
  Les comptes unifiés Bybit ne sont pas encore entièrement supportés. Pour
  trader sur Bybit avec OctoBot, il est nécessaire d'utiliser un compte
  standard. Vous pouvez facilement passer d'un compte unifié à standard en
  utilisant des sous-comptes. Transférer des fonds entre les sous-comptes est
  instantané et gratuit.
:::
