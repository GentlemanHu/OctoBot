Coinbase is a RestExchange adaptation for Coinbase exchange using the REST API.

## OctoBot support

Select the `coinbase` exchange to trade on Coinbase using OctoBot.

## Coinbase required permissions
To trade on Coinbase with OctoBot, the following rights are required on your Coinbase API Key
- `View`
- `Trade`
